# MindBalance 
Оваа верзија е целоснa и содржи:
- македонска навигација
- фиксен чет долу десно
- вежба за дишење како посебен прозорец
- рулет за грижа за себе како посебен прозорец
- видеа за учење и смирувачка музика
- проверка на училишен стрес
- следач на расположение, сон и дневни навики
- личен преглед на добросостојба

Отвори го `index.html` во browser.
